9192/1

Native execution stays on the TLS edge.

The HTTP API, MCP facade and well-known metadata are onboarding surfaces. The product core is the 9192/1 protocol behind edge.nineoneninetwo.com.br:9443.

Native edge

edge.nineoneninetwo.com.br:9443 exposes the public TLS edge. The private backend remains loopback-only and is never exposed directly.

HTTP facade

/api/v1/* offers whitelisted JSON routes for clients that need simple HTTPS onboarding.

MCP facade

/mcp exposes a remote MCP entry for agents to discover pricebook, payment rails, quotes and receipt verification.

Trust boundary

The API does not bypass the 9192 edge. Price, execution, balance, ledger and receipt state remain owned by the 9192 server path.

Supported services

GET_PULSE, MAKE_BCT, VERIFY_BCT, invoice status, payment information, receipt fetch and receipt verification.

Endpoint: edge.nineoneninetwo.com.br:9443
