# 9192 First Tester Brief

Use this when inviting a first customer, developer, or agent operator to try 9192.

## Short invite

```text
I am opening the first public test path for 9192.

The goal is simple: run one verified machine call before any paid execution.

Start here:
https://nineoneninetwo.com.br/start

Then run:
npx @nineoneninetwo/demo --mode sandbox

That command discovers the public 9192 node, creates a free sandbox quote, accepts it, executes a test call, and verifies the returned receipt.

No funding is required for the sandbox. Paid execution is separate and requires an explicit quote plus a funded machine account.
```

## What feedback to ask for

```text
1. Did the page make it clear what to do first?
2. Did the npx command run?
3. Did the sandbox return a receipt?
4. Did receipt verification return valid=true?
5. Where did the flow feel confusing or unsafe?
```

## Success condition

```text
I ran the sandbox, got a response, received a receipt, and verified it.
```

Do not optimize for paid conversion in the first test. Optimize for trust.
